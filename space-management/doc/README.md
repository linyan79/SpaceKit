# Space Management System Documentation

## 项目概述
Space Management System 是一个基于Web的CAD系统，用于空间管理和设计。该系统使用现代Web技术栈构建，提供2D和3D视图的空间设计功能。

- [组件说明](./components.md)
- [数据结构](./data-structures.md)
- [几何算法](./geometry.md)
- [开发指南](./development-guide.md)

## 技术栈
- React + TypeScript
- Three.js 用于3D渲染
- Redux Toolkit 用于状态管理
- Vite 作为构建工具

## 主要功能
1. 2D/3D空间设计
2. 墙体绘制与编辑
3. 家具布置
4. 实时3D预览
5. 属性编辑面板
